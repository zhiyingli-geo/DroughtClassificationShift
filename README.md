# Climate Change Masks Future Drought Classification Severity
Replication code for "Climate Change Masks Future Drought Classification Severity", by Zhiying Li, Jason E. Smerdon, Richard Seager, and Justin S. Mankin.
The repository contains three folders:
- `Data/`: This folder stores processed data. Intermediate processed data is not stored in this repository due to its large size, but feel free to reach out to me if you need them.
- `Figures/`. This folder stores outputs for the figures in .pdf or .jpg formats. Some figures in pdf are not stored due to their large size, but feel free to reach out to me if you require a pdf version of any figures.
- `Scripts/`. This folder contains the python code in Jupyter notebooks used for the analysis and generation of the figures.

For any questions about the code or requests for data, please contact zl68 (at) iu (dot) edu.
